Objective
To build and evaluate Decision Tree and Random Forest classification models using the Amazon Prime Titles dataset and compare their performance.

Tools Used
- Google Colab
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Steps Performed
1. Imported the Amazon Prime Titles dataset.
2. Explored dataset structure and checked for missing values.
3. Handled missing values through preprocessing.
4. Encoded categorical features using Label Encoding.
5. Split the dataset into training and testing sets.
6. Trained a Decision Tree Classifier.
7. Evaluated the Decision Tree model using accuracy score.
8. Visualized the Decision Tree structure.
9. Applied depth control to analyze overfitting.
10. Trained a Random Forest Classifier.
11. Compared Decision Tree and Random Forest performance.
12. Analyzed feature importance.
13. Visualized important features.
14. Performed cross-validation for model reliability.

Dataset
Amazon Prime Movies and TV Shows Dataset
